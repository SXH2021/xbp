HTML

HTMLとは、ハイパーテキスト・マークアップ・ランゲージ（Hyper Text Markup Language）の略。
  • h1……/h6
  • p……/p
  • i……/i
  • b……/b
  • br
  • pre……/pre
  • div……/div
  • img src="……" /
  • ol……/ol
  • ul……/ul

CSS

（Cascading Style Sheets、カスケーディング・スタイル・シート）とは、ウェブページのスタイルを指定するための言語です。
  • font-size
  • font-weight
  • font-style: italic
  • color
  • line-height
  • letter-spacing
  • text-decoration
  • text-align
  • text-indent
  • text-shadow
  • a:link{}
  • a:hover{}
  • a:active{}
  • a:visited{}
